## Optimization Techniques Applied

### 1. Code Splitting
- Dynamic imports for heavy components
- Route-based splitting (built-in Next.js)
- Manual splits for modals/forms

### 2. Image Optimization
- `next/image` with sizes attribute
- Sharp-based pre-optimization
- WebP conversion
- Lazy loading (below fold)

### 3. CSS Strategy
- Tailwind JIT compilation
- Unused class purging
- Inline critical CSS
- Scoped component styles

### 4. JavaScript
- Tree shaking (unused exports removed)
- Minification + compression
- Modern ES syntax (smaller bundles)
- Polyfills only when needed

### 5. Caching
- Static assets (immutable)
- HTML (short-lived)
- API responses (SWR pattern for future)
